# vue-admin 
这是使用vue和element写的后台管理系统，主要包含用户管理，权限管理，商品管理，订单管理和数据统计,其中分类参数，商品分类和数据报表暂无数据，其余模块都有数据，具体效果可看demo入口(https://readream.github.io/vue-admin/dist/index.html)  

## Project setup
```
npm install
```
